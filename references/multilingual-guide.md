# Multilingual Paper Writing Guide (English + Japanese)

## Overview

This guide supports researchers writing papers in both English and Japanese, whether for international journals (English primary) or Japanese domestic journals (Japanese primary). It covers parallel writing workflows, translation strategies, and formatting conventions for both languages.

---

## Parallel Writing Workflow

### Strategy 1: English First, Then Japanese (Recommended for International Papers)

Best when targeting English-language journals with a Japanese abstract required.

```
Phase 1: Draft all sections in English
Phase 2: Internal review in English
Phase 3: Translate abstract and key sections to Japanese
Phase 4: Native check for Japanese sections
Phase 5: Final alignment check (both versions say the same thing)
```

### Strategy 2: Japanese First, Then English (For Japanese Journals or When Thinking in Japanese)

Best when the primary audience is Japanese, or when the author thinks more clearly in Japanese.

```
Phase 1: Draft all sections in Japanese
Phase 2: Internal review in Japanese
Phase 3: Translate to English (or have translated)
Phase 4: Native English check / professional editing
Phase 5: Final alignment check
```

### Strategy 3: Simultaneous Bilingual Drafting

Best for experienced bilingual writers. Write each section in your stronger language for that content type.

```
- Methods: Often easier in English (standard terminology)
- Discussion: May be easier in Japanese first (complex reasoning)
- Abstract: Write in primary language first, then translate
```

## Japanese Academic Writing Conventions

### Register: である調 (De aru Style)

Japanese academic papers use である調 (da/dearu style), NOT です・ます調 (desu/masu style).

| Context | Correct | Incorrect |
|---------|---------|-----------|
| Statement | ～である | ～です |
| Negative | ～ではない | ～ではありません |
| Past | ～であった / ～した | ～でした |
| Conjecture | ～と考えられる | ～と思います |
| Purpose | ～を目的とした | ～を目的としています |

### Common Academic Expressions

| English | Japanese Academic Expression |
|---------|------------------------------|
| We investigated... | ～について検討した |
| It is suggested that... | ～が示唆される |
| It has been reported that... | ～と報告されている |
| The purpose of this study was... | 本研究の目的は～であった |
| In conclusion... | 以上より～と結論される |
| There was a significant difference | 有意差を認めた（p < 0.05） |
| No significant difference was observed | 有意差は認められなかった |
| Further studies are needed | 今後さらなる検討が必要である |
| To the best of our knowledge | 我々の知る限り |
| The limitations of this study include | 本研究の限界として |

### 専門用語の英語併記 (English Annotation of Technical Terms)

When writing in Japanese, include the English term in parentheses at first use:

```
人工知能（artificial intelligence; AI）を用いた診断支援システムについて検討した。
深層学習（deep learning）モデルとして、畳み込みニューラルネットワーク
（convolutional neural network; CNN）を採用した。
```

Rules:
- First occurrence: full English term + abbreviation
- Subsequent occurrences: Japanese term or abbreviation only
- Well-known abbreviations (DNA, CT, MRI) may not need full expansion
- Journal-specific: check target journal's policy

## Common Translation Pitfalls

### 和文英訳 (Japanese to English) Common Errors

| Japanese Pattern | Wrong English | Correct English |
|-----------------|---------------|-----------------|
| ～が報告されている | It is reported | It has been reported / Previous studies have shown |
| 検討した | We examined | We investigated / We analyzed |
| 有用であると考えられた | It was thought to be useful | It appeared to be useful / The findings suggest utility |
| 著明な改善を認めた | We recognized remarkable improvement | Marked improvement was observed |
| 特に問題なく | Without particular problems | Without complications / Uneventfully |
| ～において | In ~ | Among / During / With regard to (context-dependent) |
| 症例を経験した | We experienced a case | We report a case / We encountered a case |
| 若干の知見を得た | We obtained some findings | Our findings suggest / We found |

### 英文和訳 (English to Japanese) Common Errors

| English Pattern | Wrong Japanese | Correct Japanese |
|----------------|----------------|------------------|
| Significantly | 著しく (meaning: remarkably) | 統計学的に有意に |
| We found that... | 我々は～を見つけた | ～であることが明らかとなった |
| Robust results | 頑丈な結果 | 頑健な結果 |
| This study demonstrates | この研究は実演する | 本研究は～を示した |
| State-of-the-art | 最先端の芸術 | 最先端の / 最新の |
| Address the issue | 問題に取り組む (too casual) | ～について検討する |

### Passive Voice Conventions

- English medical papers: Passive voice common and accepted ("Blood samples were collected")
- Japanese medical papers: Also uses passive/impersonal constructions ("採血を行った" rather than "我々は採血した")
- Avoid direct "we" (我々は) at the start of every sentence in Japanese

## Section-by-Section Translation Checklist

### Title
- [ ] English title: concise, informative, no abbreviations
- [ ] Japanese title: matches English meaning exactly
- [ ] Both titles follow target journal format
- [ ] Running title provided if required (both languages)

### Abstract
- [ ] Structure matches (Background/Methods/Results/Conclusions)
- [ ] Word count within limits for both languages
- [ ] Numbers and statistics identical in both versions
- [ ] Key terms consistent with terminology table
- [ ] No information in one version that is absent from the other

### Introduction
- [ ] Same references cited in both versions
- [ ] Research question/objective stated identically
- [ ] Background context equivalent (not just translated word-for-word)

### Methods
- [ ] Statistical methods described identically
- [ ] Ethics approval statement present in both
- [ ] Technical terms match terminology table
- [ ] Study design terminology consistent

### Results
- [ ] All numerical values identical
- [ ] Table/figure references match
- [ ] Statistical values (p-values, CIs) identical
- [ ] No results in one version missing from the other

### Discussion
- [ ] Key arguments present in both versions
- [ ] Limitations section equivalent
- [ ] Future directions consistent
- [ ] Conclusion statements aligned

### References
- [ ] Same references in both versions
- [ ] Format follows respective journal guidelines
- [ ] Japanese references include English translation of title (if required)

## Translation Tools and Their Limitations

### DeepL
- **Strengths**: Generally best quality for Japanese-English medical text
- **Weaknesses**: May not handle highly specialized terminology correctly
- **Best practice**: Use as a starting draft, then manually review every sentence
- **Caution**: Do not submit AI-translated text without thorough human review

### Google Translate
- **Strengths**: Free, handles many language pairs
- **Weaknesses**: Often produces awkward medical Japanese; literal translations
- **Best practice**: Use only for rough understanding, not for manuscript text

### ChatGPT / Claude
- **Strengths**: Can maintain context across paragraphs; can follow style instructions
- **Weaknesses**: May hallucinate content not in the original; may "improve" the original
- **Best practice**: Provide the source text and ask for translation only, then verify
- **Critical rule**: Always verify that translated text says exactly what the original says

### Manual Review (Required for All Methods)
- Native or near-native speaker review is mandatory
- Focus on: medical terminology accuracy, register appropriateness, logical flow
- Consider professional medical translation services for high-impact publications

## Japanese Journal Formatting Differences

### Common Differences from International Journals

| Aspect | International (English) | Japanese Journals |
|--------|------------------------|-------------------|
| Author names | First Last | 姓 名 (Last First) |
| Affiliations | Department, Institution, City, Country | 施設名, 部門名 |
| Abstract | English only (usually) | Japanese + English (both required) |
| Keywords | English | Japanese + English |
| Figures/Tables | English captions | Japanese captions (sometimes bilingual) |
| References | Vancouver or APA style | Journal-specific (often modified Vancouver) |
| Page format | A4 or Letter, double-spaced | Often A4, specific margins |
| Word count | Word count | 文字数 (character count) |
| Ethics statement | IRB approval number | 倫理委員会承認番号 |

### Japanese Reference Formatting

For Japanese-language references:
```
著者名. 論文タイトル. 雑誌名. 発行年;巻(号):ページ.

Example:
田中太郎, 山田花子. 人工知能を用いた画像診断の現状と展望. 日本医学雑誌. 2024;150(3):234-241.
```

For English references cited in Japanese papers:
```
Author AA, Author BB. Title of the article. Journal Name. Year;Vol(Issue):Pages.
(Same as standard Vancouver format)
```

### Bilingual Abstract Format (Common in Japanese Journals)

```
【和文抄録】
目的：[研究目的]
方法：[研究方法]
結果：[主要結果]
結論：[結論]
キーワード：[キーワード1], [キーワード2], [キーワード3]

【英文抄録 / Abstract】
Objective: [Objective]
Methods: [Methods]
Results: [Results]
Conclusions: [Conclusions]
Keywords: [keyword1], [keyword2], [keyword3]
```
