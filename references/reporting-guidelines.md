## CONSORT 2010 (RCTs) - Key Items

- [ ] Trial design (parallel, crossover, factorial)
- [ ] Participants: eligibility criteria, settings
- [ ] Interventions: details for each group
- [ ] Outcomes: primary and secondary, pre-specified
- [ ] Sample size calculation
- [ ] Randomization: sequence generation, allocation concealment
- [ ] Blinding: who was blinded
- [ ] Statistical methods
- [ ] Participant flow diagram (CONSORT flowchart)
- [ ] Baseline characteristics table
- [ ] Outcomes with effect sizes and CIs
- [ ] Harms
- [ ] Registration number (ClinicalTrials.gov etc.)

## STROBE (Observational Studies) - Key Items

- [ ] Study design in title or abstract
- [ ] Setting: locations, dates, periods
- [ ] Participants: eligibility, sources, selection
- [ ] Variables: outcomes, exposures, confounders
- [ ] Data sources / measurement
- [ ] Bias: efforts to address
- [ ] Study size: how determined
- [ ] Statistical methods: confounders, subgroups, missing data
- [ ] Participants: numbers at each stage, non-participation reasons
- [ ] Descriptive data: characteristics, follow-up
- [ ] Main results with CIs and p-values
- [ ] Limitations
- [ ] Funding source

## PRISMA 2020 (Systematic Reviews) - Key Items

- [ ] Registration (PROSPERO etc.)
- [ ] Eligibility criteria (PICOS)
- [ ] Information sources and search strategy
- [ ] Selection process
- [ ] Data extraction process
- [ ] Risk of bias assessment method
- [ ] Synthesis methods (meta-analysis if applicable)
- [ ] PRISMA flow diagram
- [ ] Characteristics of included studies
- [ ] Risk of bias results
- [ ] Synthesis results with forest plots
- [ ] Certainty of evidence (GRADE)
- [ ] Funding and COI

## CARE (Case Reports) - Key Items

- [ ] Title: "Case report" in title
- [ ] Keywords: 2-5, including "case report"
- [ ] Introduction: why this case is reportable
- [ ] Patient information: demographics, symptoms, history
- [ ] Clinical findings
- [ ] Timeline (chronological)
- [ ] Diagnostic assessment
- [ ] Therapeutic intervention
- [ ] Follow-up and outcomes
- [ ] Discussion: strengths, limitations, relevant literature
- [ ] Patient perspective (if available)
- [ ] Informed consent for publication

## How to Use

1. Identify your study type
2. Download the full checklist from the guideline website
3. Fill in the checklist as you write each section
4. Submit the completed checklist with your manuscript (many journals require it)
5. Include "This study was reported according to {GUIDELINE} guidelines" in Methods
