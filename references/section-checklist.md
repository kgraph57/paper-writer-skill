# Section Quality Checklist

## Title
- [ ] Specific (describes what was studied)
- [ ] Concise (under 15 words)
- [ ] Contains important keywords
- [ ] Does not reveal the conclusion
- [ ] No abbreviations (unless universally known)
- [ ] Matches the content of the paper

## Abstract
- [ ] Within word limit
- [ ] Structured (Background, Methods, Results, Conclusions)
- [ ] Self-contained (understandable without full text)
- [ ] All numbers match full text
- [ ] No undefined abbreviations
- [ ] No references
- [ ] Conclusion matches Conclusion section

## Introduction
- [ ] 3 paragraphs (background, gap, objective)
- [ ] Paragraph 1: establishes topic importance
- [ ] Paragraph 2: identifies knowledge gap
- [ ] Paragraph 3: states study objective clearly
- [ ] Objective matches Conclusion
- [ ] Concise (not an exhaustive review)
- [ ] All claims referenced
- [ ] No results included

## Methods
- [ ] Study design stated
- [ ] Study period specified
- [ ] Setting described
- [ ] Inclusion/exclusion criteria defined
- [ ] Variables and definitions listed
- [ ] Measurement methods described
- [ ] Statistical methods specified
- [ ] Software and version stated
- [ ] Ethics approval stated
- [ ] Informed consent addressed
- [ ] Reproducible (another researcher could replicate)
- [ ] Every method has a corresponding result

## Results
- [ ] Follows same order as Methods
- [ ] No interpretation or opinion
- [ ] No references to other studies
- [ ] All Tables/Figures mentioned in text
- [ ] Numbers consistent with Tables/Figures
- [ ] Appropriate statistics (CI, p-values)
- [ ] Every result has a corresponding method
- [ ] Patient flow described (enrolled, excluded, analyzed)

## Discussion
- [ ] Paragraph 1: main findings summary
- [ ] Compares with prior literature
- [ ] Does not introduce new results
- [ ] No excessive speculation
- [ ] Claims supported by references
- [ ] Limitations section present and honest
- [ ] Clinical implications stated
- [ ] Focused (not too long)

## Conclusion
- [ ] Answers the Introduction objective
- [ ] One core message
- [ ] Brief (1-2 sentences)
- [ ] No new data
- [ ] No references
- [ ] Consistent with Abstract conclusion

## References
- [ ] All cited papers are in reference list
- [ ] All references are cited in text
- [ ] Formatted per journal style
- [ ] DOIs included (if required)
- [ ] Recent and relevant
- [ ] No retracted papers

## Figures & Tables
- [ ] Each has a descriptive caption/legend
- [ ] Each mentioned in text
- [ ] Figures at 300+ dpi
- [ ] Tables not duplicating figure data
- [ ] Consistent units and decimal places
- [ ] Color-blind friendly (if color used)

## Overall
- [ ] Word count within journal limits
- [ ] Reporting guideline followed (CONSORT/STROBE/PRISMA/CARE)
- [ ] No plagiarism
- [ ] Consistent terminology throughout
- [ ] Author contributions defined
- [ ] Conflicts of interest disclosed
- [ ] Funding acknowledged

## Case Report Specific Checklist

### Case Presentation
- [ ] Patient demographics included (age, sex)
- [ ] Chief complaint clearly stated
- [ ] Relevant medical history documented
- [ ] Clinical findings described (both positive and negative)
- [ ] Timeline of events clear and chronological
- [ ] Diagnostic workup described with results
- [ ] Diagnostic reasoning explained
- [ ] Treatment described with dosing
- [ ] Treatment rationale explained
- [ ] Follow-up duration and outcomes reported
- [ ] Patient perspective included (if available)
- [ ] Informed consent for publication documented

### Case Report Introduction
- [ ] Background on the condition is brief
- [ ] Clearly states why this case is reportable
- [ ] Does not reveal outcome prematurely
- [ ] 2-3 paragraphs maximum

### Case Report Abstract (CARE format)
- [ ] Background states why case is reportable
- [ ] Patient demographics included
- [ ] Diagnosis clearly stated
- [ ] Treatment and outcome summarized
- [ ] Clinical lesson stated in conclusion
- [ ] Does NOT use Methods/Results structure

### Case Report Title
- [ ] Contains "case report"
- [ ] Includes diagnosis or key finding
- [ ] Contains relevant keywords

### CARE Checklist Items
- [ ] Title includes "case report"
- [ ] Keywords include "case report"
- [ ] Patient perspective included (if available)
- [ ] Informed consent for publication obtained
- [ ] Timeline clearly presented
- [ ] Diagnostic reasoning explained
- [ ] All relevant investigations reported with results
- [ ] Treatment rationale explained
- [ ] Follow-up duration and outcomes stated
- [ ] Limitations of the case discussed
