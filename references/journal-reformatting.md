# Journal Reformatting Quick Guide

> After rejection, reformatting for the next journal should take hours, not days. This guide identifies what typically needs changing.

---

## Reformatting Checklist

### Always Change

- [ ] **Cover letter** — New editor name, new journal-specific content
- [ ] **Reference format** — Vancouver vs APA vs NLM vs numbered
- [ ] **Abstract format** — Restructure headings per new journal requirements
- [ ] **Word counts** — Trim or expand to new limits
- [ ] **Figure/table count** — Move excess to Supplementary if over limit
- [ ] **Title page format** — Running head length, author format
- [ ] **Reporting checklist** — Same guideline but journal's preferred format/version

### Sometimes Change

- [ ] **Section headings** — Some journals use non-standard headings
- [ ] **Conclusion** — Some journals require separate Conclusion; others forbid it
- [ ] **Key Points / Highlights** — Add or remove per journal requirement
- [ ] **Data availability statement** — Wording may differ
- [ ] **Ethics statement placement** — Methods vs end of paper vs separate
- [ ] **Figure format** — TIFF vs EPS vs PNG; DPI requirements

### Rarely Change

- Content of Methods, Results, Discussion (unless reviewer feedback prompted revision)
- Study data or analysis
- Author list

## Common Format Differences

### Reference Styles

| Style | Example | Journals |
|-------|---------|---------|
| **Vancouver (numbered)** | "...infection rates.¹" or "...infection rates [1]." | NEJM, JAMA, Lancet, BMJ, most medical |
| **APA (author-date)** | "...infection rates (Smith et al., 2024)." | Psychology, public health, some nursing |
| **NLM** | Similar to Vancouver with variations | PubMed-indexed journals |
| **Harvard** | "(Smith 2024)" | Some UK/Australian journals |
| **CSE** | Citation-name or citation-sequence | Science/biology journals |

### Abstract Heading Variations

| Heading | JAMA | NEJM | Lancet | BMJ |
|---------|------|------|--------|-----|
| Purpose | "Importance" | "Background" | "Background" | "Objective" |
| Methods | "Design, Setting, and Participants" | "Methods" | "Methods" | "Design/Setting/Participants" |
| Results | "Results" | "Results" | "Findings" | "Results" |
| Conclusion | "Conclusions and Relevance" | "Conclusions" | "Interpretation" | "Conclusions" |
| Extra | Key Points (separate) | Funded by | Funding | "What this paper adds" |

### File Naming Conventions

| Portal | Preferred File Names |
|--------|---------------------|
| ScholarOne | "Main_Document.docx", "Figure_1.tiff" |
| Editorial Manager | "Manuscript.docx", "Fig1.eps" |
| eJournalPress | Follow specific naming guide |
| General | Use descriptive names without spaces |

## Quick Reformatting Steps

### Step 1: Document Current Format (5 min)
```
Current journal: [name]
Reference style: [Vancouver/APA/etc.]
Abstract format: [headings list]
Word limit: [N]
Figure/table limit: [N]
Special sections: [Key Points, Highlights, etc.]
```

### Step 2: Document Target Format (10 min)
```
Target journal: [name]
Reference style: [Vancouver/APA/etc.]
Abstract format: [headings list]
Word limit: [N]
Figure/table limit: [N]
Special sections: [Key Points, Highlights, etc.]
```

### Step 3: Execute Changes (1-3 hours)
1. Reformat references (use reference manager — Zotero, EndNote, Mendeley)
2. Restructure abstract with new headings
3. Adjust word count (trim Discussion first if over)
4. Add/remove special sections (Key Points, Highlights)
5. Reformat title page
6. Write new cover letter
7. Update reporting checklist version if needed
8. Recheck figure specifications

### Step 4: Verify (30 min)
- [ ] All references renumbered correctly
- [ ] Abstract within new word limit
- [ ] Main text within new word limit
- [ ] All figures meet new DPI/format requirements
- [ ] Cover letter addresses correct editor
- [ ] No mention of previous journal name anywhere in manuscript

## Reference Manager Tips

### Switching Styles Quickly

| Tool | How to Switch |
|------|--------------|
| **Zotero** | Edit → Preferences → Cite → change style; refresh document |
| **EndNote** | Format Bibliography → change output style |
| **Mendeley** | References → Change Citation Style |
| **Paperpile** | Paperpile menu → Citation style → search and select |

### Manual Reference Reformatting

If no reference manager is used:
1. Export references to a .bib or .ris file
2. Import into free tool (Zotero is recommended)
3. Apply new citation style
4. Copy formatted references

## Cascading Submission Strategy

Plan your cascade BEFORE first submission:

```
Journal 1 (Reach):    NEJM / JAMA / Lancet
  ↓ desk reject (1-2 weeks)
Journal 2 (Target):   Specialty top journal
  ↓ peer review reject (6-8 weeks)
Journal 3 (Safety):   Solid mid-tier journal
  ↓ if still rejected
Journal 4 (Backup):   Open access / lower tier
```

Track submissions:

| Journal | Submitted | Decision | Turnaround | Action |
|---------|-----------|----------|-----------|--------|
| NEJM | 2024-01-15 | Desk reject | 10 days | Reformat for JACC |
| JACC | 2024-01-28 | Major revision | 45 days | Revise and resubmit |
| JACC (R1) | 2024-04-15 | Accepted | 20 days | — |

## 日本語

### リフォーマットの要点

- 参考文献スタイルの変更が最も時間がかかる → 文献管理ソフトを使用
- 抄録の見出し変更は必須
- カバーレターは毎回新規作成
- 前のジャーナル名が原稿内に残っていないか確認（「submitted to [前ジャーナル名]」等）
- 日本語→英語ジャーナル、英語→日本語ジャーナルの切り替えは大幅な書き直しが必要
