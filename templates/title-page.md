# Title Page Template

> The title page is required by virtually all medical journals. This template covers elements specified by ICMJE Recommendations (January 2026 update).

---

## Template

```
[ARTICLE TYPE: Original Article / Case Report / Review / Letter / etc.]

TITLE:
[Full descriptive title — max 15-20 words, suitable for electronic retrieval]

SHORT TITLE / RUNNING HEAD:
[Max 40 characters including spaces]

AUTHORS:
1. [First name] [Middle initial]. [Last name], [Degree(s)]^{1,2} [ORCID: 0000-0000-0000-0000]
2. [First name] [Middle initial]. [Last name], [Degree(s)]^{2} [ORCID: 0000-0000-0000-0000]
3. ...

AFFILIATIONS:
1. [Department], [Institution], [City], [State/Province], [Country]
2. [Department], [Institution], [City], [State/Province], [Country]

CORRESPONDING AUTHOR:
[Full name], [Degree(s)]
[Department], [Institution]
[Full postal address]
Email: [email]
Phone: [number]
ORCID: [ID]

WORD COUNT:
- Abstract: [N] words
- Main text: [N] words (excluding abstract, references, figure legends, tables)
- References: [N]
- Tables: [N]
- Figures: [N]
- Supplementary files: [N]

KEYWORDS:
[Keyword 1]; [Keyword 2]; [Keyword 3]; [Keyword 4]; [Keyword 5]
(3-10 terms; use MeSH terms where possible)

CLINICAL TRIAL REGISTRATION:
[Registry name: ClinicalTrials.gov / UMIN-CTR / jRCT / ISRCTN / etc.]
[Registration number: NCT00000000]
[URL: https://clinicaltrials.gov/study/NCT00000000]

PRIOR PRESENTATIONS:
[Conference name, date, city — if data were previously presented]

FUNDING SOURCES:
[Brief summary — detailed statement in Declarations section]

DISCLAIMERS:
[If applicable]
```

## Notes

### ORCID
- **Required** for corresponding author at most major journals (ICMJE recommendation since 2024)
- **Encouraged** for all authors
- Free registration: https://orcid.org/register
- Format: `https://orcid.org/0000-0000-0000-0000`

### Running Head Rules by Journal
| Journal | Max Characters | Notes |
|---------|--------------|-------|
| NEJM | 50 | Typically auto-generated |
| JAMA | 40 | Required on title page |
| The Lancet | Not required | |
| BMJ | 50 | Short title required |
| Most Elsevier | 50 | Running title field in portal |

### Degree Abbreviations (Common in Medical Papers)
- MD, PhD, MPH, MSc, BSc, DrPH, MBBS, DO
- List highest relevant degree(s) only
- Some journals: no degrees listed (check instructions)

### Blinding for Double-Blind Review
For double-blind journals, prepare two versions:
1. **Title page** (with all author information) — uploaded separately
2. **Blinded manuscript** (no author names, affiliations, or identifying information in headers/footers)
