# Pre-Submission Checklist

## Manuscript Files

- [ ] Title page (title, authors, affiliations, corresponding author, word count)
- [ ] Abstract (within journal word limit)
- [ ] Main manuscript (all sections in reading order)
- [ ] References (formatted per journal style)
- [ ] Tables (each on a separate page, with titles and footnotes)
- [ ] Figure legends (separate list)
- [ ] Figures (high resolution: 300+ dpi for print, appropriate format)
- [ ] Supplementary materials (if any)

## Content Checks

- [ ] All co-authors have reviewed and approved the final manuscript
- [ ] Abstract numbers match the full text exactly
- [ ] All Tables/Figures are cited in the text
- [ ] Methods and Results correspond 1:1
- [ ] Introduction objective matches Conclusion answer
- [ ] No new results in Discussion
- [ ] No interpretation in Results
- [ ] Reference list is complete (every citation has a reference, every reference is cited)
- [ ] Consistent terminology throughout (no synonym cycling)

## Journal-Specific Requirements

- [ ] Word count within limit: _____ / _____ words
- [ ] Abstract word count within limit: _____ / _____ words
- [ ] Number of references within limit: _____ / _____
- [ ] Number of Tables within limit: _____ / _____
- [ ] Number of Figures within limit: _____ / _____
- [ ] Citation style correct (Vancouver / APA / other: _____)
- [ ] Reporting guideline followed: _____ (CONSORT / STROBE / PRISMA / CARE / other)
- [ ] Reporting guideline checklist completed and ready to upload

## Formatting

- [ ] Double-spaced (if required)
- [ ] Line numbers added (if required)
- [ ] Page numbers present
- [ ] Headers/footers per journal requirements
- [ ] Font and size per journal requirements

## Ethics & Declarations

- [ ] Ethics approval statement included (IRB name, approval number)
- [ ] Informed consent statement included
- [ ] Conflict of interest disclosure for ALL authors
- [ ] Funding sources disclosed
- [ ] Author contributions defined (CRediT or ICMJE format)
- [ ] Data availability statement included (if required)
- [ ] Clinical trial registration number (if applicable)

## AI Writing Check

- [ ] Humanizer pass completed (Phase 4)
- [ ] No AI vocabulary remaining
- [ ] Sentence rhythm varies naturally
- [ ] No generic positive conclusions
- [ ] Consistent academic tone throughout

## Submission Materials

- [ ] Cover letter prepared
- [ ] Suggested reviewers (2-3 names with emails, if required)
- [ ] Opposed reviewers noted (if any)
- [ ] ORCID IDs for all authors
- [ ] Graphical abstract (if required)
- [ ] Highlights / Key Points (if required)
