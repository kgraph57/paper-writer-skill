# Conclusion Writing Guide

## Structure

1-2 sentences maximum. Contains:

1. **Result summary**: What did you find? (one sentence)
2. **Implication**: What does it mean clinically? (one sentence, optional)

## Rules

- Must directly answer the research objective stated in Introduction Paragraph 3
- One core message only
- No new information
- No references
- No speculation
- Be direct and specific

## Template

> "In this {study design}, {Factor X} was {found to be / identified as} {an independent predictor of / associated with} {Outcome Y} in patients with {Population}. {Optional: Clinical implication in one sentence.}"

## Quality Check

| Check | Pass? |
|-------|-------|
| Answers the research question from Introduction? | |
| One core message? | |
| No new data or references? | |
| Consistent with Abstract conclusion? | |
| Brief (1-2 sentences)? | |

## Common Mistakes

- Overstating findings ("This will revolutionize treatment")
- Multiple messages competing for attention
- Including data not in Results
- Not matching Introduction objective
