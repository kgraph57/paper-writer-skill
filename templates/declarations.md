# Declarations Section Templates

Use these templates for the Declarations section of your manuscript. Most journals require all of these statements. Check your target journal's requirements.

---

## Ethics Approval

### Template (Clinical Study)

```
This study was approved by the [Institution Name] Institutional Review
Board / Ethics Committee (approval number: [NUMBER], date: [DATE]).
The study was conducted in accordance with the Declaration of Helsinki
(2013 revision).
```

### Template (Retrospective, Consent Waiver)

```
This study was approved by the [Institution Name] Institutional Review
Board (approval number: [NUMBER]). The requirement for informed consent
was waived due to the retrospective nature of the study and the use of
de-identified data.
```

### Template (No Human Subjects)

```
This study did not involve human subjects and was exempt from
institutional review board approval.
```

### Japanese

```
本研究は[施設名]倫理委員会の承認を得て実施した（承認番号：[番号]、
承認日：[日付]）。ヘルシンキ宣言（2013年改訂）に従い実施した。
```

---

## Informed Consent

### Template (Prospective Study)

```
Written informed consent was obtained from all participants prior to
enrollment in the study.
```

### Template (Retrospective, Opt-Out)

```
Due to the retrospective nature of this study, the requirement for
individual informed consent was waived. An opt-out notice was posted
at the institution in accordance with ethical guidelines.
```

### Template (Case Report)

```
Written informed consent was obtained from the patient [and/or legal
guardian] for publication of this case report and accompanying images.
A copy of the consent form is available for review by the Editor.
```

### Japanese

```
全対象者から書面によるインフォームドコンセントを取得した。

（後ろ向き研究の場合）
本研究は後ろ向き研究であり、オプトアウト方式により同意を取得した。

（症例報告の場合）
本症例報告および画像の公表について、患者から書面による同意を取得した。
```

## Conflict of Interest

### Template (No Conflicts)

```
The authors declare no conflicts of interest relevant to this work.
```

### Template (With Conflicts)

```
[Author Name] has received [consulting fees / research grants /
speaker fees] from [Company Name]. [Author Name] serves on the
advisory board of [Company Name]. All other authors declare no
conflicts of interest.
```

### Japanese

```
著者らに開示すべき利益相反はない。

（利益相反ありの場合）
[著者名]は[企業名]より[コンサルタント料/研究助成金/講演料]を受領
している。その他の著者に開示すべき利益相反はない。
```

## Funding

### Template (Funded)

```
This work was supported by [Funding Agency] [grant type] [grant number]
(to [PI Name]). The funder had no role in study design, data collection,
analysis, interpretation of data, writing of the manuscript, or the
decision to submit for publication.
```

### Template (No Funding)

```
This research did not receive any specific grant from funding agencies
in the public, commercial, or not-for-profit sectors.
```

### Japanese

```
本研究は[助成機関名]（課題番号：[番号]）の助成を受けた。助成機関は
研究デザイン、データ収集・解析、論文執筆、投稿の意思決定に関与して
いない。

（資金なしの場合）
本研究に対する特定の助成はない。
```

## Data Availability

### Template (Open Data)

```
The datasets generated and/or analyzed during the current study are
available in the [Repository Name] repository, [URL/DOI].
```

### Template (Available on Request)

```
The datasets generated and/or analyzed during the current study are
available from the corresponding author on reasonable request.
```

### Template (Restricted)

```
The datasets generated and/or analyzed during the current study are
not publicly available due to [patient privacy / institutional policy]
but are available from the corresponding author on reasonable request
with appropriate ethics approval.
```

### Japanese

```
本研究で使用したデータは[リポジトリ名]（[URL]）で公開されている。

（要請に応じて提供の場合）
本研究のデータは、合理的な要請があれば責任著者より入手可能である。
```

### Template (English)

```
During the preparation of this manuscript, the authors used [Tool name,
version] ([Company]) for [specific purposes]. The authors reviewed,
edited, and verified all AI-assisted output and take full responsibility
for the content of this publication.
```

### Template (Japanese)

```
本論文の作成にあたり、著者らは[ツール名]（[企業名]）を[目的]の
目的で使用した。AI支援による出力は著者らが確認・編集・検証しており、
本論文の内容について全責任を負う。
```
