# Proof Correction Guide

> After acceptance, the corresponding author receives galley proofs (page proofs) for final review before publication. This is the LAST opportunity to correct errors.

---

## Timeline
- Proofs arrive: typically 2-8 weeks after acceptance
- Turnaround required: **24-72 hours** (strict deadline)
- Late corrections may delay publication or be rejected

## What to Check

### Critical (Must Check)
- [ ] Author names and affiliations — correct spelling, order, superscript numbers
- [ ] Abstract — all numbers match the main text
- [ ] Tables — all data values correct, no transposition errors
- [ ] Figures — correct figures in correct positions, acceptable quality
- [ ] Reference list — all references present, correct numbering, no broken links
- [ ] Corresponding author email — correct
- [ ] ORCID iDs — linked correctly
- [ ] Funding and COI statements — accurate
- [ ] Clinical trial registration number — present and correct
- [ ] DOI and citation — verify early view/online first citation format

### Important (Should Check)
- [ ] Headers and footers — correct running head
- [ ] Page numbers — sequential
- [ ] Line spacing and formatting — acceptable
- [ ] Supplementary material links — functional
- [ ] Copyright/license statement — correct (especially for OA)

### NOT Allowed at Proof Stage
- Rewriting sentences or paragraphs
- Adding new data, references, or authors
- Changing conclusions or interpretation
- Extensive reformatting

## How to Submit Corrections

### PDF Annotation Method (Most Common)
1. Open proof PDF in Adobe Acrobat or Preview
2. Use sticky notes or text annotations for corrections
3. For each correction: state page, column, line, and exact change
4. Return annotated PDF via journal's proofing system

### Correction List Method
```
Page X, Line Y: Change "incorrect text" to "correct text"
Page X, Table 2, Row 3, Column 4: Change "15.3" to "15.8"
Page X, Figure 2 legend: Add "CI, confidence interval" to abbreviation list
```

### Common Proofing Systems
| System | Journals |
|--------|----------|
| Proof Central (Wiley) | Wiley journals |
| CATS (Elsevier) | Elsevier journals |
| eProofing (Springer Nature) | Nature, Springer journals |
| Direct PDF return | Many other journals |
